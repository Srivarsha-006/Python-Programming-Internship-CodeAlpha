# 📂 Projects

## 🎮 Task 1: Hangman Game

A console-based Hangman game where the player guesses a randomly selected word one letter at a time.

### Features
- Random word selection
- Maximum of 6 incorrect guesses
- Tracks guessed letters
- Win/Lose conditions

**Concepts Used**
- Random Module
- Loops
- Conditional Statements
- Strings
- Lists

## 📈 Task 2: Stock Portfolio Tracker

A simple portfolio tracker that calculates the total investment value using predefined stock prices.

**Concepts Used**
- Dictionaries

- File Handling
